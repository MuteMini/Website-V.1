Arrays & Arraylists

Arrays store multiples of variables

To put it clearly, Arrays give a way to store many different instances of a variable type into one variable name. They are consecutive memory locations that hold values of the same data type.

Image from Java Docs

Each location is stored at an index, which in turn is called an array's element. An array starts at index 0.

To access an index, you use square brackets, and inside put the index number you want to access.

//creates and sets the length of the array.
int[] array = new int[3];

//sets the values of the array.
array[0] = 13;
array[1] = 56;
array[2] = -2;

//accesses the index 1 of the int array.
System.out.println( array[1] );

You can get the length of the array using:

array.length

Arrays can store Arrays

This is called a multidimentional array. You can think of the most basic multidimentional array as a matrix or a grid, filled with the variable type declared.

Now, the multidimentional array uses multiple indices, each referring to a different section of the array. You could think of multidimentional arrays like arrays that contain arrays that contain arrays, so on and so forth.

Image from HWS Colleges

Lets say you have a two dimentional array like:

//creates and sets the length of the array.
int[][] array = new int[3][4];

To access an element on the first array, and inside that array the fourth element, you would type:

System.out.println( array[1][3] );

ArrayLists, and it's flexibility

An Arraylist is a flexible and resizeable array. You can not modify the length of the arraylist, but you can add and remove elements easily.

To create an Arraylist in Java and manipulate it, you use:

import java.util.ArrayList; //imports the class ArrayList for use

ArrayList<Integer> arraylist = new ArrayList<Integer>();

arraylist.add(7); //adds the integer 7 to index 0 of the arraylist [7]
arraylist.add(2); //adds the integer 2 to index 1 of the arraylist [7, 2]
arraylist.add(5); //adds the integer 5 to index 2 of the arraylist [7, 2, 5]

arraylist.remove(0); //removes the integer at index 0. [2, 5]
arraylist.remove(0); //removes the integer at index 0. [5]

arraylist.set(0, 100) //alters the integer at index 0 to be 100 [100]

System.out.println( arraylist.size() ); //prints the length of the arraylist. 1

arraylist.clear(); //removes everything in the arraylist. []

To change the type of variable the Arraylist holds, change the object inside < >.

//a String arraylist
ArrayList<String> stringList = new ArrayList<String>();

//a Double arraylist
ArrayList<Double> doubleList = new ArrayList<Double>();

//an arraylist that stores objects that are an instance of Custom
ArrayList<Custom> CustomList = new ArrayList<Custom>();
